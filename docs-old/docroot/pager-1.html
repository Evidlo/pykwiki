Themes - November 18, 2013

Built-in themes

PyKwiki features some built-in themes that can be installed with pykwiki theme install <theme_name>.

Default

The default PyKwiki theme.

Features

  • Valid HTML5
  • Mobile friendly
  • Uses Bootstrap 3
  • Supports searching
  • Includes respond.js for IE8 support
  • Includes mustache.js for javascript templates
  • Supports GitHub, Facebook, and Twitter: like, follow, watch.

Amelia

A colorful theme based on Amelia from Bootswatch.

Features

  • Valid HTML5
  • Mobile friendly
  • Uses Bootstrap 3
  • Supports searching
  • Includes respond.js for IE8 support
  • Built-in support for Github, Facebook, and Twitter social buttons

Barebones

A minimal stripped starter theme without CSS, designed to be used as a template for new themes.

Features

  • Infinitely nested menus
  • Valid HTML5
  • Ready for customization

  • Supports searching

Webserver - November 18, 2013

Webserver Configuration

Apache

To configure Apache to work with PyKwiki, you'll need a virtual host entry for static files that points to the PyKwiki install.

For example, if your project directory is /home/hpotter/pykwiki/MyBlog, then a virtual host entry might appear as follows:

<VirtualHost *:80>
    ServerName www.myblog.com
    DocumentRoot "/home/hpotter/pykwiki/MyBlog/docroot/"
    ErrorDocument 404 /404.html
    <Directory "/home/hpotter/pykwiki/MyBlog/docroot/">
        RewriteEngine Off
        Options Indexes FollowSymLinks MultiViews
        AllowOverride None
        Order allow,deny
        allow from all
    </Directory>
</VirtualHost>
